HTML

¿Que es HTML?

HTML es el codigo estandar utilizado para estructurar y desplegar paginas web y sus contenidos como parrafos, imagenes, enlaces y tablas

Historia HTML

Fue creado por TIM Berners en 1991 en el CERN para facilitar el intercambio de documentos cientificos mediante hipervinculos

HTML 5

HTML 5 es la ultima version de HTML, antes pasaron por ejeplo por la version 3 o la 4.
